# 鞋类产品展示网站

一个简洁专业的鞋类产品展示网站，采用现代化设计风格，支持动态管理、自动排版和产品详情弹窗展示。

## 项目结构

```
网站文件/
├── index.html              # 网站首页/产品列表页（动态加载）
├── about.html              # 关于我们页面
├── contact.html            # 联系方式页面
├── admin.html              # 配置管理页面
├── style.css               # 全局样式文件
├── app.js                  # 首页逻辑（动态加载产品、弹窗功能）
├── common.js               # 通用脚本（多语言、配置管理）
├── admin.js                # 管理页面逻辑
├── README.md               # 项目说明文档
└── images/
    ├── 1-平底系列/         # 平底系列产品图片
    ├── 2-真皮系列/         # 真皮系列产品图片
    └── 3-短靴系列/         # 短靴系列产品图片
```

## 功能特点

- **产品展示**: 分系列展示产品，每款产品配有图片和详细规格
- **产品详情弹窗**: 点击产品卡片弹出详情，无需跳转页面
- **灯箱查看**: 点击任意图片查看大图，支持ESC关闭
- **多角度展示**: 自动加载相关图片（整体图、侧视图、俯视图等）
- **产品规格**: 鞋面材质、内里材质、鞋底材质、起订量、交货周期
- **响应式设计**: 适配桌面端、平板和移动设备
- **现代化UI**: 采用简洁优雅的设计风格
- **页面导航**: 包含产品列表、关于我们、联系方式三个页面
- **配置管理**: 可视化管理界面，支持多标签页管理
- **多语言切换**: 支持中文/英文双语切换
- **本地存储**: 配置保存在浏览器localStorage中

## 技术栈

- HTML5
- CSS3 (Grid布局 + Flexbox)
- 原生JavaScript (ES6+)
- LocalStorage API

## 使用方法

### 快速开始

1. 将项目文件上传到Git仓库或服务器
2. 确保图片路径正确
3. 直接在浏览器中打开 `index.html` 即可查看

### 查看产品详情

**弹窗详情页：**
1. 在首页产品列表中点击任意产品卡片
2. 弹窗显示产品大图、规格信息、相关图片
3. 点击主图或缩略图可查看灯箱大图
4. 点击弹窗外部或按ESC关闭弹窗

**多角度图片展示：**
产品详情弹窗会自动尝试查找并显示以下相关图片：
- 主图（原始图片）
- 整体图：`系列10整体.jpg`
- 侧视图：`系列10侧视图.jpg`
- 俯视图：`系列10俯视图.jpg`
- 细节图：`系列10细节.jpg`
- 后视图：`系列10后视图.jpg`

**注意**：相关图片需要实际存在于对应文件夹中才会显示。

### 管理网站

访问 `admin.html` 进入配置管理页面：

#### 基本设置
- 网站名称
- 页面标题
- 页面副标题
- 页脚文本

#### 产品管理
- 点击"+ 添加系列"创建新的产品系列
- 编辑系列名称
- 为每个产品设置：
  - 产品名称
  - 产品描述
  - 图片路径
  - 鞋面材质
  - 内里材质
  - 鞋底材质
  - 起订量
  - 交货周期
- 点击"+ 添加图片"添加新产品
- 删除不需要的系列或图片
- 点击"💾 保存配置"保存所有修改

#### 关于我们
- 公司简介
- 企业理念（每行一条，使用" - "分隔标题和描述）

#### 联系方式
- 公司地址
- 联系电话
- 电子邮箱
- 工作时间（支持HTML）

### 多语言切换

- 点击导航栏右上角的"中文"或"EN"按钮
- 页面内容自动切换语言
- 语言设置保存在浏览器本地

### 添加新产品图片

**简单方式（推荐）**:

1. 将新图片上传到对应的系列文件夹：
   - `images/1-平底系列/` - 平底系列
   - `images/2-真皮系列/` - 真皮系列
   - `images/3-短靴系列/` - 短靴系列

2. 打开 `admin.html`
3. 点击"+ 添加图片"添加新产品
4. 填写产品信息和图片路径

**添加新系列**:

```
步骤：
1. 创建新文件夹 images/4-新系列/
2. 将该系列的所有图片放到该文件夹
3. 打开 admin.html
4. 点击 "+ 添加系列"
5. 修改系列名称为"新系列"
6. 点击 "+ 添加图片" 添加多张图片
7. 保存配置
```

### 添加多角度产品图片

为了在产品详情弹窗显示多角度图片，请按照以下命名规则添加图片：

```
主图: 系列10.jpg
整体图: 系列10整体.jpg
侧视图: 系列10侧视图.jpg
俯视图: 系列10俯视图.jpg
细节图: 系列10细节.jpg
后视图: 系列10后视图.jpg
```

将这些图片都放在对应的系列文件夹中，系统会自动识别并显示。

## 注意事项

1. **图片路径**: 确保图片路径正确，推荐使用相对路径 `images/1-平底系列/`
2. **文件夹命名**: 系列文件夹名称必须为 `1-系列名` 格式
3. **本地存储**: 配置保存在浏览器本地，清除浏览器数据会丢失配置
4. **自动加载**: 首页会自动从localStorage加载配置并渲染产品
5. **图片格式**: 支持jpg、png、webp等常见图片格式
6. **响应式**: 网站会自动适配不同屏幕尺寸，产品网格会自动调整列数
7. **多角度图片**: 相关图片需要实际存在于文件夹中才会显示

## 常见问题

**Q: 配置丢失了怎么办？**
A: 点击管理页面的"🔄 重置配置"恢复默认设置

**Q: 如何在不同设备同步配置？**
A: 由于配置存储在本地，不同设备需要分别配置。建议在完成配置后，将 `localStorage` 中的配置导出为JSON文件进行备份。

**Q: 新添加的图片不显示？**
A: 检查图片路径是否正确，图片文件是否存在于指定位置

**Q: 如何删除某个系列？**
A: 在管理页面的"产品管理"选项卡中找到对应系列，点击"删除"按钮

**Q: 多角度图片不显示？**
A: 检查相关图片文件是否存在，文件名是否符合命名规则

**Q: 如何查看产品大图？**
A: 点击产品卡片打开详情弹窗，点击主图或任意缩略图即可查看大图

**Q: 灯箱怎么关闭？**
A: 点击灯箱空白区域、点击X按钮或按ESC键即可关闭

**Q: 多语言切换后内容不变？**
A: 检查浏览器控制台是否有错误，确保common.js正确加载

**Q: 弹窗点击无效？**
A: 检查app.js是否正确加载，确保没有JavaScript错误

**Q: 配置管理页面显示异常？**
A: 清除浏览器localStorage，刷新页面重置配置
